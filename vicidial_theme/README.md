# Vicidial Bootstrap Theme

Ce thème fournit des interfaces modernes pour Vicidial à l'aide de Bootstrap 5.
Il inclut maintenant un exemple de backend PHP très simple pour la connexion et la navigation.

## Installation

1. Copier le dossier `vicidial_theme` dans `/var/www/html/vicidial/`.
2. Accéder à `login.php` pour se connecter.
3. Les fichiers `css/theme.css` (clair) et `css/theme-dark.css` (sombre) gèrent les styles.
4. Identifiants d'exemple :
   - **admin@example.com** / **adminpass** (rôle admin)
   - **agent@example.com** / **agentpass** (rôle agent)
5. Les pages principales :
   - `index.php` : tableau de bord admin
   - `campaigns.php` : gestion des campagnes
   - `users.php` : liste des utilisateurs
   - `reports.php` : rapports
   - `agent.php` : interface agent

Ce backend est uniquement démonstratif et ne couvre pas toutes les fonctionnalités de Vicidial.
